모둠 말하기 활동

원인과 결과 이어말하기

장소 경험을 뽑고, 원인과 결과를 차례로 이어 말해 봅시다.

모둠

입력 방법

모둠원 이름과 경험 장소를 입력하세요. 경험은 장소 중심으로 짧게 씁니다. 예: 운동장, 급식실, 교실, 도서관, 놀이터, 학원, 집, 공원, 병원, 체육관

이어말하기 판

1. 경험 장소 뽑기

경험 버튼을 눌러 주세요.

2. 원인 또는 결과 두 번 뽑기

1차: 기다리는 중
2차: 기다리는 중

1차 뽑기 뒤, 2차 뽑기는 반대 내용으로 나옵니다.

3. 알맞은 연결어 확인하기

1차 연결어 선택
2차 연결어 선택
원인은 “왜냐하면 ~ 때문입니다”, 결과는 “그래서”를 사용합니다.

4. 동료평가

친구가 경험 장소에 맞게 원인과 결과를 바르게 이야기했으면 OK 버튼을 눌러 주세요.
